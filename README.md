# Проект: Место

## Описание проекта и функциональность
Данный проект - сервис Mesto: интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.

### Технологии
В данном проекте используются технологии flex и grid-выравнивания, создание сайта по макету Figma, адаптивность сайтов, технологии оптимизации шрифтов, базовые технологие java script (использование простейших функций, основы DOM, template-элементы, работа с массивами ).
